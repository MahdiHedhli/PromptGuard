## Code / artifact highlights (blog post fodder)

- **The judge "shipped but not validated" decision is honest in a way the security audience will appreciate.** A frequent failure mode of v1 launches is "we shipped the feature, here are the wiring tests, go forth." Day-9's diagnostic explicitly distinguishes wiring from detection. Wiring is proved (latency delta). Detection is not (model output is malformed). The DEC-019 narrative is the right shape for a CISO-readable post-mortem section in the launch blog.
- **Cross-category-overlap-as-annotation-gap** is a methodology nuance worth a sidebar. Our regex layer found a domain inside a postgres URL. The synth corpus marked only the URL as expected, so the domain detection counts as a false positive in our F1 numbers. The detector is right; the annotation is incomplete. Naming this clearly in the methodology table teaches readers to read F1 numbers carefully.
- **Native Ollama vs docker-Ollama on Apple Silicon: 61 tok/s vs 0.28 tok/s.** A 220x deployment difference because Docker on macOS does not pass through the Metal GPU. Worth a deployment-note paragraph in the operator docs and the blog's "what we learned in production" section.
- **Audit log no-text fuzz invariant** runs 200 random PII strings and asserts none appear in log content. The audit log is structurally guaranteed not to leak the text it was created to protect against. This is the kind of test that reads well in a security-tooling blog post.
